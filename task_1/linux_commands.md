# Linux Commands Used

## pwd

Used to check the current working directory.

Example:

```bash
pwd
```

---

## ls

Used to list files and folders in the current directory.

Example:

```bash
ls
```

---

## ls -la

Used to list all files, including hidden files.

Example:

```bash
ls -la
```

---

## cd

Used to move between directories.

Example:

```bash
cd task_1
```

---

## mkdir

Used to create new directories.

Example:

```bash
mkdir data
mkdir src
```

---

## touch

Used to create a new empty file.

Example:

```bash
touch sample.txt
```

---

## nano

Used to create and edit files from the terminal.

Example:

```bash
nano README.md
```

---

## cat

Used to display the contents of a file.

Example:

```bash
cat requirements.txt
```

---

## echo

Used to print text to the terminal.

Example:

```bash
echo "Hello World"
```

---

## cp

Used to copy files.

Example:

```bash
cp sample.txt sample_copy.txt
```

---

## mv

Used to move or rename files.

Example:

```bash
mv sample_copy.txt temp.txt
```

---

## rm

Used to delete files.

Example:

```bash
rm temp.txt
```

---

## grep

Used to search for specific text in a file.

Example:

```bash
grep "numpy" requirements.txt
```

---

## find

Used to search for files and folders.

Example:

```bash
find task_1
```

---

## head

Used to view the first few lines of a file.

Example:

```bash
head requirements.txt
```

---

## tail

Used to view the last few lines of a file.

Example:

```bash
tail requirements.txt
```

---

## wc

Used to count lines, words, and characters in a file.

Example:

```bash
wc requirements.txt
```

---

## chmod

Used to change file permissions.

Example:

```bash
chmod +x hello.py
```

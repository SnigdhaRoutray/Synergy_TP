# Task 2 - Python Recap Assignment

## Objective

Read student submission data from a CSV file and generate a JSON summary report.

## Files

- src/analyzer.py
- src/main.py
- data/submissions.csv
- output/summary.json

## Requirements

Install dependencies:

pip install -r requirements.txt

## Run

From the repository root:

python task_2/src/main.py task_2/data/submissions.csv task_2/output/summary.json

## Output

The program generates a summary report in:

task_2/output/summary.json
